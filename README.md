## 使用

**生成路径中不要包含中文和空格，并且生成过程中尽量关闭所有的杀毒软件，否则会生成失败！**

64位免杀效果远好于32位，能用64尽量用64（注意对应的 shellcode 也需要是64位）。

**针对Cobalt Strike，不要选择生成Windows分阶段木马、Windows无阶段木马，而是生成payload，最终是一个payload.c文件。**
